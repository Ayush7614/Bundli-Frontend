About GoTo Blogs Saver

GoTo Blog Saver is built to save time and blogs of bloggers.

It is a web-app that give its users privilege to save all their ideas. All the posts are saved in our database and the user can access them anytime for the further usage. Useful links for ideas, images and other popular blogging websites are also provided for quick access.


The Front-end of this web-app is built using HTML, CSS and BootStrap. The Back-end relies completely on the awesome Python-Flask. It is basically a fun, Flask-Learn-Time Project developed and maintained by an aspiring Software Developer Sushant Gaurav.
